AI Text to Speech Extension

This page is part of the AI Text to Speech browser extension. If you're seeing this page, it means the extension is installed, but you need to interact with it through the browser extension interface.

To use the extension:

  1. Select text on any webpage
  2. Click the floating button that appears
  3. Or right-click and choose "Convert to speech" from the context menu
  4. Configure settings by clicking the extension icon in your browser toolbar
